ngx-dynamic-table-component Demo

The ngx-dynamic-table-component is a versatile Angular component that allows you to dynamically generate tables using Bootstrap. You can provide data in various formats to create tables with customizable headers, footers, and cell styles.

Using string[][]

You can provide a string[][] array to generate a simple table. For example: 


  [
    ['Nombre', 'Apellido', 'Edad'],
    ['Juan', 'Gómez', '45'],
    ['Pedro', 'Dominguez', '36'],
    ['Alberto', 'Pérez', '27'],
    ['Promedio', '', '36']
  ]

Using TableOptions

For more advanced configurations, you can provide a TableOptions object. This allows you to customize headers, footers, and individual cell styles, and enable sorting on specific columns. For example: 

        
  {{'{'}}
    header: [
      {{'{'}} text: 'Nombre', sortable: true },
      {{'{'}} text: 'Apellido', sortable: false },
      {{'{'}} text: 'Edad', sortable: true }
    ],
    body: [
      [
        {{'{'}} text: 'Juan', bold: true },
        {{'{'}} text: 'Gómez' },
        {{'{'}} text: '45', italic: true }
      ],
      [
        {{'{'}} text: 'Pedro' },
        {{'{'}} text: 'Dominguez', class: 'custom-class' },
        {{'{'}} text: '36' }
      ],
      [
        {{'{'}} text: 'Alberto' },
        {{'{'}} text: 'Pérez' },
        {{'{'}} text: '27' }
      ]
    ],
    footer: [
      [
        {{'{'}} text: 'Promedio' },
        {{'{'}} text: '' },
        {{'{'}} text: '36' }
      ]
    ],
    class: 'table-class'
  }
